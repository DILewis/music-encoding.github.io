Step three: the setup of a single <measure> with an empty <staff> and <layer>.

Within <score>, the <section> element is the container for the <measure> elements. It's positioned directly after <scoreDef>.

Measures can contain multiple staves (<staff>) which can have multiple layers (<layer>). The <layer> element will be the parent element for the <note> elements in the next step.

To indicate the numbered position of measures, staves or layers, the attribute @n can be used.

In the editor, insert a <section> element. Inside <section> add an empty <measure> element. Set the number of the pickup measure to "0". Add a child element <staff> to the <measure> element and set its number to "1" (first staff in measure 0). Then add an unnumbered child element <layer> to the <staff> element. Again, your input will automatically be inserted at the correct position right after the closing statement of </scoreDef>.
