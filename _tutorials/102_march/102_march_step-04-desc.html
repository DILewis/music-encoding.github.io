In this step you will learn encode a single note.

Please encode the first note of the example. Here is an example of how to encode a whole note C4: <note pname="c" oct="4" dur="1"></note>.

In <note> element you can define single note characteristics by using the following attributes:

  • pitch name @pname – “c”, “d”, “e”, “f”, “g”, “a” or “b”</li>
  • octave @oct – pitch’s octave notated in scientific pitch notation, namely “0”-”9”</li>
  • duration @dur – e.g. “1” for whole note, “2” for half note, “4” for quarter note, “8” for eighth note, “16” for sixteenth note, … “2048” for 2048th note
