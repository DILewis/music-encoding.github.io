In this step you will learn the application and use of namespaces.

As mentioned in the previous step, the name of the root element of an XML document is basically an arbitrary name that has no meaning in itself. To make clear in which context an element is defined and to avoid element name conflicts, XML makes use of so called namespaces. Namespaces can be declared by adding a @xmlns namespace declaration to the root element.

The MEI namespace is: http://www.music-encoding.org/ns/mei
