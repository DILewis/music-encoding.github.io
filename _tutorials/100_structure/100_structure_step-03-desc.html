In this step you will learn about the main child elements of the <mei> root element.

At a minimum, a valid MEI-encoded file contains two structures within the parent <mei> element: the <meiHead> and <music> elements.

The <meiHead> structure contains elements that describe the work (i.e., metadata), including information about authorship, encoding standards, and provenance.

The <music> structure contains information regarding the encoded music itself. The music notation is represented using XML tags, arranged in a hierarchical relationship.
